# Debt Counter
https://debt-count.herokuapp.com/
## Aim: 
Helps any employee from secretary to manager with or without legal or accounting knowledge to correctly calculate arrears and penalties for presentation to the debtor
## Used technologies: 
###### Backend: 
* Java 14, 
* Spring Boot, 
* REST
###### Frontend:
* Angular 9, 
* Angular Material, 
* SCSS, 
* Bootstrap 4
